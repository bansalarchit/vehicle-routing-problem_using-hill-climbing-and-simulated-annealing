the code use two text file 
1.15citydataset - contain coordinate of 9 customers
2.vrpdataset - contain coordinate of 48 customers

reference for datset https://people.sc.fsu.edu/~jburkardt/datasets/tsp/tsp.html

uses matplotlib package to draw scatter plot

the code demand for two input at console 

first for the type of algorithm that one wanted to run
enter 1 for hill climbing with random restart
enter 2 for hill climbing with random walk
enter 3 for simulated annealing

second for the tpe of dataset that one want to use
enter 1 for 9 customers data set
enter 2 for 48 customers data set

at console give command python project.py in order to run the code
